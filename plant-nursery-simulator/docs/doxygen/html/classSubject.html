Subject::~Subject ( )
virtual

Virtual destructor that notifies observers of shutdown.

Emits a SubjectDestroyed event with a standard shutdown message before clearing subscriptions. Observers must not assume the subject remains valid after the call; they should drop any cached pointers immediately.
